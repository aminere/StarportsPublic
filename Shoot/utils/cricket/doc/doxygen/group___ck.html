Function Documentation

bool CkGetClipFlag ( )

Returns the value of the clip flag.

The clip flag is set to true whenever the final audio output "clips", i.e. exceeds the maximum value.

float CkGetRenderLoad ( )

Returns a rough estimate of the load on the audio processing thread.

This is the fraction of the interval between audio callbacks that was actually spent rendering audio; it will range between 0 and 1.

void CkLockAudio ( )

Lock the audio processing thread to prevent it from processing any API calls.

The lock should be held only for a short time. This is useful to ensure that certain calls are processed together. For example, if you want to play multiple sounds together exactly in sync, call CkLockAudio(), call play() on each sound, then call CkUnlockAudio().

void CkResetClipFlag ( )

Resets the value of the clip flag to false.

The clip flag is set to true whenever the final audio output "clips", i.e. exceeds the maximum value.
